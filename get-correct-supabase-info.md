# 正しいSupabase情報の取得方法

## 手順:

1. Supabaseダッシュボードで `gijiroku-system` プロジェクトを開く

2. 左サイドバーから **Settings** (⚙️) をクリック

3. **API** をクリック

4. 以下の情報をコピーしてください：

```

```
