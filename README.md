# 会議議事録管理システム

全社ナレッジ共有プラットフォーム

## 🌐 本番環境

- **URL**: https://gijiroku-web.pages.dev （仮）
- **カスタムドメイン**: https://gijiroku.ielove-partners.jp （設定予定）

## 🔐 認証システム

### 初回セットアップ
- システム初回起動時に管理者アカウントを作成
- メール確認（@ielove-partners.jp ドメイン制限）
- パスワード強度チェック

### ユーザー登録
- 社員はメールアドレスで登録申請
- 管理者が承認してアクセス許可
- アルバイト等は登録拒否可能

### ログイン
- メールアドレスとパスワードでログイン
- ログイン情報は自動記憶（次回から自動入力）
- アカウント停止機能あり

## 👥 ユーザー管理

### ユーザーステータス
- **pending**: 承認待ち
- **active**: アクティブ（ログイン可能）
- **suspended**: 停止中（ログイン不可）
- **rejected**: 承認拒否

### ユーザー役割
- **admin**: 管理者（ユーザー管理可能）
- **user**: 一般ユーザー
- **viewer**: 閲覧者

### 管理者機能
- ユーザーの承認・拒否
- アカウントの停止・有効化
- ユーザーの削除
- アクセスログの記録

## 🛡️ セキュリティ

- ドメイン制限（@ielove-partners.jp のみ）
- パスワード強度チェック（8文字以上、大文字・小文字・数字）
- メール確認による本人確認
- ログイン試行のログ記録
- アカウント停止機能

## 📊 機能

### 組織階層管理
- ボード（経営層）
- 部門
- チーム

### 会議管理
- 会議の作成・編集・削除
- 会議情報の管理

### 議事録管理
- 議事録の作成・編集・削除
- 8項目の詳細情報管理
  - 議題・アジェンダ
  - 具体的なアクション
  - 担当者
  - 期限（カレンダー選択可能）
  - 目的・期待される効果
  - ステータス（4種類）
  - 備考1・備考2

### ステータス管理
- 議事録全体: 3ステータス（会議未実施 / 会議終了（残タスクあり） / 会議終了（残タスクなし））
- 個別タスク: 4ステータス（未着手 / 進行中 / 完了 / 期限超過）

### AI解析機能
- テキストから議事録を自動生成
- ファイルアップロード対応

### エクスポート機能
- PDF出力
- Word出力
- Excel出力
- CSV出力
- TXT出力
